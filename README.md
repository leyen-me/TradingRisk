# 长桥证券日内期权风控系统

## 适用

日内期权，不持仓过夜，降低风险。

## 入场

利用TradingView的警报和指标、策略系统发出买卖，做多、做空信号。(SuperTrend、 RSI)等指标

## 止盈止损

开仓后立即设置止盈止损，止损线默认是最近20根K线的最小值（相关行情需要购买OpenApi），如果低于该价格，则平仓

止盈设置为2.5%固定平仓
